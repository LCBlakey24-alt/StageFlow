# StageFlow V1 Build Plan

## Locked direction

StageFlow is a generic, framework-based, timetable-first assessment app.

## Core flow

1. Build/import timetable.
2. Open lesson.
3. Add learners if needed.
4. Take register.
5. Assess present learners.
6. Save lesson.
7. Generate reports, certificates and end-of-term packs.

## Next build order

1. Supabase project
2. Auth
3. Organisation accounts
4. Database schema
5. Replace local storage with database
6. File storage for logos/certificates
7. Real export generation
8. Staging and live deployment
